# SQLWareHouse
This is my attempt of creating a SQL Warehouse using SQL Server to consolidate data,analytical reporting and decision making.

1.Data Source-we are importing data from two datasets(ERP and CRM) as CSV files.
2.Data Quality - Cleanse and resolve data before analysis.
3.Integration - Combine both datasets into a single, user friendly data model.
4.Scope - Focus on latest dataset, no historization.
5.Documentation - Provide clear documentation to data model to provide good insights.
